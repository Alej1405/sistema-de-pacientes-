# TurnosPaciente

Proyecto Java para la gestión de turnos de pacientes (Estructura de Datos).

Ubicación local (fuera de iCloud) para evitar conflictos de sincronización.

## Estructura

```
TurnosPaciente/
├── src/        Código fuente (.java)
├── bin/        Clases compiladas (.class)
└── .vscode/    Configuración del proyecto (Java 25)
```

## Compilar y ejecutar

```bash
javac -d bin src/Main.java
java -cp bin Main
```
